HTML <div>

Description: The <div> element is a generic container used to group together other HTML elements or sections of content. It's commonly used for styling purposes and structuring web layouts.

Example: <div class="container">...</div>

HTML <h1>

Description: The <h1> element is a heading element representing the most important heading on the page. It's typically used for the main title or headline.

Example: <h1>Welcome to My Website</h1>

HTML <h2>

Description: The <h2> element is a heading element representing a level 2 heading on the page. It's used for subheadings or titles of sections within the content.

Example: <h2>About Us</h2>

HTML <hr>

Description: The <hr> element is used to create a thematic break or horizontal line to visually separate content within a webpage. It's often used to indicate a change in topic or section.

Example: <hr>

HTML class

Description: The class attribute is used to specify one or more class names for an HTML element. Classes are used to apply CSS styles or JavaScript behavior to one or more elements.

Example: <div class="container">...</div>
